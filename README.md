# optimisation-multiclass-regression

## Project Description
Implementation of multiclass logistic regression solved using:
- Gradient Descent (GD)
- Block Coordinate Gradient Descent with Gauss-Southwell rule (BCGD)

## Dataset
Forest Cover Type Dataset (Kaggle/UCI)
- 581,012 samples, 54 features, 7 classes

## Files
- 'homework optimization.ipynb' - main Jupyter notebook with all code and results

## Requirements
- numpy
- scipy
- sklearn
- matplotlib

## How to Run
1. Open the notebook in Google Colab
2. Run all cells sequentially

## Results
- GD accuracy: 0.7159
- BCGD accuracy: 0.7138
- Baseline (DummyClassifier): 0.4846
